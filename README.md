# EM_Wave_Project

Семестровый проект С++

Тема: визуалиция электромагнитной волны в виде анимации колебания двух синусоид (электрического поля - красный цвет и магнитного поля - синий цвет) во взаимно 
перпендикулярных плоскостях.

Сборка: чтобы собрать проект, необходимо склонировать репозиторий, а затем прописать в терминале "make run". После закрытия программы требуется очистить объектные файлы, прописав в терминале "make clean".

Управление: 
- клавишами стрелочек можно менять угол обзора
- клавишами "q" и "e" можно ускорять и замедлять волну соответсвенно
- клавишей "пробел" можно поставить на паузу или выйти из этого состояния

Польза: наглядная демонстрация поведения модели электромагнитной волны, которая может использоваться в школах при знакомстве учеников с данной темой.
